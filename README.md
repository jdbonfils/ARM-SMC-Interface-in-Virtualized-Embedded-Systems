# ARM-SMC-Interface-in-Virtualized-Embedded-Systems
Study of the ARM SMC Interface in Virtualized Embedded Systems

The goal of this research project was to study how the SMC interface is handled in virtualized systems across different hypervisors on ARM platforms and identify the resulting attack surface. This study led to the discovery of security flaws in Xen (Zynq) and Jailhouse (i.MX) and two security vulnerabilities in the firmware of Zynq UltraScale MPSoC platforms (CVE-2025-48507, CVE-2025-0038).
